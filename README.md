⚡ AI Smart Energy Management System

An AI-powered system to predict electricity usage, detect anomalies, and provide smart recommendations.

 🚀 Features
- 🔮 Time-series forecasting using Prophet
- 🚨 Anomaly detection using Isolation Forest
- 💰 Electricity bill prediction
- 💡 Smart recommendations engine
- 📊 Interactive dashboard using Streamlit
- ⚙️ Usage simulation & optimization

  Live Demo
👉 https://smartenergyai-xxoexv38cdowsqb6nuyd2k.streamlit.app/

 🛠 Tech Stack
- Python
- Streamlit
- Prophet
- Scikit-learn
- Pandas
